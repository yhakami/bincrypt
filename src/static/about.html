About BinCrypt

Browser-side encrypted pastebin for short-lived text sharing

What is BinCrypt?

BinCrypt is a pastebin for short-lived text sharing. If you set a password, encryption happens in your browser before upload and the server stores ciphertext.

No-password pastes are intentionally stored server-side as unencrypted paste data. Use that mode only when anyone with the link can read the content.

Privacy Policy

Data Collection

We collect minimal data:

  • Encrypted paste content for password-protected pastes
  • Unencrypted paste content when you choose no-password mode
  • Paste metadata (creation time, expiration date)
  • No personal information is collected
  • No cookies or tracking

Data Retention

Pastes are automatically deleted after their expiration period (default: 7 days). Deleted data is permanently removed and cannot be recovered.

Security

Password-protected paste content is encrypted client-side using AES-256-GCM before transmission. No-password paste content is protected by link access, expiration, and moderation controls.
