Enhanced Zero-Knowledge Architecture

2024-01-15

We've implemented a new zero-knowledge proof system that ensures even we cannot decrypt your pastes. All encryption now happens client-side with military-grade AES-256-GCM encryption before data leaves your browser.

This update includes improved key derivation using PBKDF2 with 100,000 iterations and cryptographically secure random salt generation for each paste.

Security Feature

Premium Features Now Available

2024-01-10

Introducing BinCrypt Premium with extended features for power users. Premium subscribers get 90-day paste expiration (vs 7 days), 5MB file size limit (vs 512KB), and priority support.

We accept Bitcoin payments through BTCPay Server for maximum privacy. No personal information required - just paste, pay, and encrypt.

Feature Update

Burn After Read Implementation

2024-01-05

New "Burn After Read" feature automatically deletes your paste after the first view. Perfect for sharing sensitive information that should only be accessed once.

Combined with password protection, this provides an additional layer of security for your most sensitive data.

Syntax Highlighting for 18+ Languages

2024-01-01

Enhanced code sharing with syntax highlighting support for JavaScript, Python, Go, Rust, HTML, CSS, JSON, Bash, TypeScript, Java, C++, C#, PHP, Ruby, YAML, SQL, and Markdown.

All highlighting is processed client-side to maintain our zero-knowledge principles.
